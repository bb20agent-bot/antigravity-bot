# 🚀 바이낸스 자동 거래 봇 완벽 설치 가이드

이 가이드는 트레이딩뷰(TradingView) 웹훅을 받아 바이낸스(Binance) 선물(Futures) 시장에서 자동으로 매매하고, 그 거래 결과를 실시간으로 구글 시트(Google Sheets)에 기록하는 완벽한 봇 설정 방법입니다.

## 1️⃣ 구글 시트 세팅 (GoogleSheetLogger.gs)
1. 구글 스프레드시트를 새로 생성합니다.
2. 첫 번째 행(A1~G1)에 컬럼 제목을 적어줍니다: `시간`, `종목`, `액션`, `가격`, `수량`, `손익`, `메시지`
3. 상단 메뉴에서 **확장 프로그램 > Apps Script**를 클릭합니다.
4. 열린 스크립트 편집 창에 제공된 **`GoogleSheetLogger.gs`** 파일의 코드를 전체 복사하여 붙여넣고 저장합니다.
5. 우측 상단의 파란색 **[배포(Deploy)] -> [새 배포]**를 클릭합니다.
6. 유형 선택(톱니바퀴)에서 **웹 앱(Web App)**을 선택합니다.
7. 액세스 권한을 반드시 **"모든 사용자(Anyone)"**로 설정하고 배포합니다.
8. 화면에 생성된 **웹 앱 URL (Webhook URL)**을 복사해 둡니다.

---

## 2️⃣ 바이낸스 API 키 발급
1. 바이낸스 로그인 후 우측 상단 프로필 > **API Management**로 이동합니다.
2. **Create API** (System generated)를 눌러 새 키를 생성합니다.
3. API Restrictions에서 **Enable Futures(선물 거래 활성화)** 권한을 체크합니다.
4. 🚨 보안을 위해 **IP Access Restrictions**에 봇이 실행될 PC나 서버의 고정 IP 주소를 입력하는 것을 강력히 권장합니다.
5. **API Key** 와 **Secret Key** 를 절대 외부로 유출되지 않는 안전한 곳에 복사해 둡니다.

---

## 3️⃣ 환경 변수 설정 (.env)
1. 제공된 `.env.example` 파일의 이름을 `.env`로 변경합니다. (윈도우에서는 이름 바꾸기 시 `.env.` 처럼 끝에 점을 찍으면 쉽게 바꿀 수 있습니다)
2. 메모장 등 텍스트 편집기로 열어 발급받은 키 값들을 입력합니다.
```env
BINANCE_API_KEY=당신의_바이낸스_API키
BINANCE_API_SECRET=당신의_바이낸스_시크릿키
GOOGLE_SHEET_WEBHOOK=아까_복사해둔_구글앱스스크립트_URL
PORT=5000
```

---

## 4️⃣ Python 봇 설치 및 실행
1. 시스템에 Python 3.8 이상이 설치되어 있는지 확인합니다.
2. 터미널(명령 프롬프트, PowerShell, VSCode 터미널)을 열고, 봇 파일이 있는 폴더경로로 이동(`cd 명령어`)합니다.
3. 봇 실행에 필요한 필수 라이브러리를 설치합니다:
   ```bash
   pip install -r requirements.txt
   ```
4. 모든 설치가 완료되면 스크립트를 실행합니다:
   ```bash
   python binance_trading_bot.py
   ```
5. 터미널 화면에 `Running on http://0.0.0.0:5000/` (혹은 비슷한 메시지)가 뜨면 프로그램이 정상적으로 신호를 기다리고 있다는 뜻입니다.

---

## 5️⃣ 트레이딩뷰(TradingView) 알람 웹훅 설정
트레이딩뷰의 지표나 전략 얼러트(Alert) 생성 창에서 다음과 같이 설정합니다.

- **Webhook URL 체크**: `http://(봇이 실행되는 서버의 IP 주소):5000/webhook` 
  - *참고: ngrok 같은 포트포워딩 툴을 쓰면 로컬 PC에서도 외부 접속 URL을 발급받아 사용할 수 있습니다.*
- **메시지 (Message)**: 반드시 아래와 같은 **JSON 구조 형식**으로 입력해야 파이썬 봇이 정확히 인식합니다.

```json
{
  "symbol": "BTC/USDT",
  "action": "buy",
  "quantity": 0.01,
  "sl_percent": 1.0,
  "tp_percent": 2.0
}
```

### 💡 파라미터 설명 (Message 내부 값)
* **`symbol`**: 코인 심볼 (바이낸스 ccxt 기준 `BTC/USDT`, `ETH/USDT` 등 슬래시 포함 규격)
* **`action`**: 구매 포지션은 `"buy"` (롱 진입), 판매 포지션은 `"sell"` (숏 진입) 
* **`quantity`**: 매수/매도할 코인 수량 (바이낸스 최소 주문 단위를 충족해야 함)
* **`sl_percent`** (옵션): 주문가 대비 강제 손절 라인 비율 (1.0 = 1% 손절)
* **`tp_percent`** (옵션): 주문가 대비 강제 익절 라인 비율 (2.0 = 2% 익절)
