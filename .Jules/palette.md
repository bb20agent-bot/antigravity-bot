## 2024-03-01 - [Vora App Navigation ARIA Labels and Copy Link Feedback]
**Learning:** Found that the app heavily uses icon-only buttons in its bottom navigation, which lacked accessible labels. Implemented a robust `CopyLinkButton` with visual feedback to confirm clipboard action, which significantly improves user experience and clarity for the "Your Fandom Link". Also needed to gracefully handle environments where the Clipboard API might be missing, adding an `execCommand` fallback. Stubbing imports to avoid build issues for missing files requires careful replacement of the import statements themselves to avoid reference errors and ensure successful compilation.
**Action:** When adding interaction to utility components like copy buttons, always provide immediate visual feedback (e.g., swapping icons) and ensure a robust fallback exists for older or restrictive environments (like some Telegram mini-apps). Always double-check that local stubs accurately mimic and replace the exact missing import statement to prevent build crashes.
