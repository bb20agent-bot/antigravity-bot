# VORA VIP 트레이딩뷰 웹훅 텔레그램 알림 설정

## 개요
TradingView의 웹훅 기능을 활용하여 특정 조건(RSI 다이버전스) 만족 시 텔레그램 채널(https://t.me/voraflywheel)로 자동 알림을 전송하는 시스템을 구축합니다.

## 대상 종목
- **BTCUSD** (비트코인)
- **ETHUSD** (이더리움)
- **XAUUSD** (금)

## 조건 (VORA VIP 시그널)
- **타임프레임 (기준봉)**: 1시간봉 / 15분봉 / 5분봉
- **트리거 조건**: RSI DIVERGENCE 지수 **+17 이상** 또는 **-17 이하**일 때 알람 발생

## 텔레그램 메세지 포맷 (예시)
```text
🚨 [VORA VIP 시그널] 진입 타점 포착!

📌 종목: BTCUSD
⏱️ 기준: 15분봉
🎯 신호: 지수 +17 상향 돌파 , 진입  대기 : 지수 20돌파 레드 신호 진입
💵 현재가: 77,960.5
```
